# Changelog

All notable changes to the Forest Mist theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-22

### Added
- Initial release of Forest Mist theme
- Main Forest Mist theme variant with full contrast
- Forest Mist Soft variant with reduced contrast
- Complete color scheme based on nature-inspired palette
- Support for all major programming languages
- Semantic highlighting support
- Bracket pair colorization
- Terminal color customization
- Git decoration colors
- Comprehensive UI theming for all VS Code elements

### Features
- **Pine Green** (#5f8a5f) - Primary accent color for keywords and UI elements
- **Moss Green** (#8fbc8f) - String literals and success states
- **Sky Blue** (#87ceeb) - Functions, links, and information
- **Earth Brown** (#cd853f) - Constants, numbers, and warnings
- **Mist Blue** (#b0c4de) - Types and secondary elements
- **Bark Brown** (#8b7355) - Operators and language variables

### Design Principles
- Minimum 4.5:1 contrast ratio for optimal readability
- Color temperature optimized for reduced eye strain
- Moderate saturation levels to prevent visual fatigue
- Scientific approach to color selection for productivity

### Supported Languages
- JavaScript / TypeScript
- Python
- HTML / CSS / SCSS / LESS
- JSON / YAML / XML
- Markdown
- React / Vue / Angular
- Node.js
- PHP
- Java / C# / C++
- Go / Rust
- And many more!

### Technical Details
- Full semantic highlighting support
- Bracket pair colorization with 6 distinct colors
- 16 ANSI terminal colors
- Complete UI theming (activity bar, sidebar, panels, etc.)
- Git integration with status colors
- Error, warning, and info highlighting
- Search and selection highlighting
- Diff editor support
